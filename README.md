## 📚 프로젝트 개요 
🫅팀구성 및 역할
- 구본홍 : 축제 CRUD, 축제 달력, 상세페이지, 게시판 CRUD, 즐겨찾기 기능  
- 김승재 : 마이 페이지(댓글 및 즐겨찾기 연동), 회원 관리 CRUD, 로그인/ 로그아웃, 메인 홈 화면  
- 지문수 : 지도 SVG, 관광 CRUD, 상세페이지, 댓글 기능, 요구사항 정의서, PPT 제작

## 📌 프로젝트 기능
+ 메인 페이지
  - 인기 광광지 및 축제 정보 제공
    
+ 회원기능
  - 회원 가입, 로그인/로그아웃
  - 마이 페이지 (회원정보 수정, 탈퇴, 비밀번호 변경)
  - 즐겨찾기한 목록 조회
  - 작성 게시글, 댓글 조회
    
+ 관리자 기능
  - 회원 정보 조회 및 삭제
  - 축제/관광지 관리
    
+ 축제/관광지 관리
  - 축제/관광지 (등록, 수정, 삭제)
  - 기간별 / 지역별 축제 조회
    
+ 게시판 기능
  - 사용자 게시판(작성/수정/삭제)
  - 관리자 게시판(작성/수정/삭제)
    
+ 댓글 및 즐겨찾기
  - 댓글 작성/삭제
  - 즐겨찾기 기능
